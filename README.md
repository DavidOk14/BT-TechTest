Mini project I was working on when I found out SDL3 had dropped with Vulkan support.

Running the executable "BT-TechTest.exe" is aimmed to print out the window size and position (By default centered at 1280x720)
The executable will also print SDL/Vulkan based arguments and your best supported GPU alongside the name of it and brand
